Level 1 Hints and Knowledge Review

1. Name the movement.

In a basic hinge joint, if the joint angle is decreasing, this is known as flexion.

If the joint angle is increasing, this is known as extension.

Check the model again to see what is happening.

2. Name the contraction type.

If the movement is going against gravity, or lifting weight, it is concentric.

If the movement is going with gravity or lowering weight, it is eccentric.

Check the model again to see what is happening.
